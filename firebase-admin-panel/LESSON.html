MI Trade Master — Lesson Generator

One-time tool: fetches your existing courses, asks Groq to write ordered topic lessons for each, and publishes them straight to Firestore. Run locally in a browser — nothing here is saved anywhere except your own Firebase project.

1. Connection Status

Connecting automatically...

2. Groq API Keys

Used only in your browser to call Groq directly — never sent anywhere else. If Key 1 hits a rate limit, the tool automatically switches to Key 2 (and back) instead of stopping.

3. Select Chapters to Generate

4. Generate & Publish

For each selected chapter: existing lessons are listed first (so nothing is duplicated), then new topic lessons are generated to fill the requested count, in order, and written to courses/{courseId}/lessons.

Ready. Connect Firebase, add your Groq key, select chapters, then run.
